1. Crea una variable llamada "nombre" y asígnale tu nombre como valor. Luego, imprime en la consola "Hola, [nombre]".

2. Crea una variable llamada "edad" y asígnale un número como valor. Luego, utiliza un condicional para imprimir "Eres mayor de edad"
 si la edad es mayor o igual a 18, o "Eres menor de edad" si la edad es menor que 18.

3. Crea un arreglo llamado "numeros" y asígnale 5 números enteros como valores. Luego, utiliza un for para recorrer el arreglo y calcular
 la suma de todos los elementos. Imprime el resultado en la consola.

4. Crea una función llamada "multiplicar" que reciba dos números como parámetros y devuelva el resultado de multiplicarlos. 
Luego, utiliza la función para calcular el resultado de 3 x 5 y 7 x 9.

5. Crea un arreglo llamado "nombres" y asígnale varios nombres como valores. Luego, utiliza un for para recorrer el arreglo y 
imprimir cada nombre en la consola.

6. Crea una función llamada "esPar" que reciba un número como parámetro y devuelva true si es par o false si es impar. 
Luego, utiliza la función para comprobar si 2, 5 y 10 son pares.

7. Crea un arreglo llamado "edades" y asígnale varias edades como valores. 
Luego, utiliza un for para recorrer el arreglo y calcular la media de todas las edades. Imprime el resultado en la consola.

8. Crea una función llamada "esPrimo" que reciba un número como parámetro y devuelva true si es primo o false si no lo es. 
Luego, utiliza la función para comprobar si 7, 10 y 13 son primos.

9. Crea un arreglo llamado "frutas" y asígnale varias frutas como valores. 
Luego, utiliza un for para recorrer el arreglo y imprimir cada fruta en la consola, pero solo si la primera letra de la fruta es una "p".

10. Crea una función llamada "esPalindromo" que reciba una palabra como parámetro y devuelva true si es un palíndromo o false si no lo es. 
Luego, utiliza la función para comprobar si "oso", "casa" y "reconocer" son palíndromos
